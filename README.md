# gh_action_demo
action使用示例


# 使用限制 
+ https://docs.github.com/zh/actions/administering-github-actions/usage-limits-billing-and-administration
+ 公开的仓库是免费的
+ 每次(publish可以生成)生成的机器可以用35天，1个job可以用6小数
+ 每个账号限制并发作业数量是20
+ 若要更多作业，就需要更多账号
+ 目前不能指定ip所在的地区
